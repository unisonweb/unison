data Name Source #

A name is an absolute-or-relative non-empty list of name segments. It is used to represent the path to a definition.

A few example names:

  • "foo.bar" --> Name Relative ("bar" :| ["foo"])
  • ".foo.bar" --> Name Absolute ("bar" :| ["foo"])
  • "|>.-- Name Relative (":| ["|"])
  • "." --> Name Relative ("." :| [])
  • ".." --> Name Absolute (".." :| [])
