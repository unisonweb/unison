Description

Type class for IntervalMap keys.

As there is no sensible default, no instances for prelude types are provided (E.g. you might want to have tuples as closed intervals in one case, and open in another).

Empty intervals, i.e. intervals where 'lowerBound >= upperBound' should be avoided if possible. If you must use empty intervals, you need to provide implementations for all operations, as the default implementations do not necessarily work correctly. For example, the default implementation of inside returns True for a point that is equal to the lowerBound of a left-closed interval even if it is larger than the upper bound.
