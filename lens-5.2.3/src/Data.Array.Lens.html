-- | This t'Setter' can be used to derive a new 'IArray' from an old 'IArray' by
-- applying a function to each of the indices to look it up in the old 'IArray'.
--

-- @
-- 'ixmap' ≡ 'over' '.' 'ixmapped'
-- 'ixmapped' ≡ 'setting' '.' 'ixmap'

-- 'bounds' ('over' ('ixmapped' b) f arr) ≡ b
-- @
