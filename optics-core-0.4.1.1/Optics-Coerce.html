Optics.Coerce

Description

This module defines operations to coerce the type parameters of optics to a representationally equal type. For example, if we have

newtype MkInt = MkInt Int

and

l :: Lens' S Int

then

coerceA @Int @MkInt l :: Lens' S MkInt
